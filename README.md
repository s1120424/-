# 基於 RISC-V 架構之多功能電子密碼鎖系統 (實作於 Basys3 開發板)
本專案為元智大學 114-2 EEB318A「數位系統設計與實驗」課程之期末專題成果。系統利用硬體描述語言 Verilog 於 Xilinx Artix-7 FPGA (Basys3) 開發板上建構一個具備 RISC-V 微處理器核心（PicoRV32）的計算機系統，並透過記憶體映射 I/O (Memory-Mapped I/O, MMIO) 技術控制指撥開關、按鈕、LED 與七段顯示器，實現具備高度靈活性與安全性之電子密碼鎖。

## 📌 系統功能特點
* **待機模式 (0000)**：系統上電或按下重製鍵後回到初始狀態，七段顯示器即時同步指撥開關之 16-bit 密碼輸入。
* **驗證成功 (PASS)**：當輸入密碼與系統預設密碼 (`16'h1234`) 相符並按下確認鍵時，七段顯示器鎖定顯示「PASS」，綠色 LED 常亮。
* **驗證失敗 (Err)**：當輸入密碼不符並按下確認鍵時，七段顯示器顯示「Err」，紅色 LED 規律閃爍以示警報。
* **系統硬體重製**：按下板上 Reset 按鈕後，處理器 PC 清零重啟，系統回歸待機狀態。

## 🗺️ 記憶體映射位址規劃 (Memory Map)
系統將外部硬體周邊暫存器映射至特定的 32-bit 定址空間，由處理器透過指令進行讀寫：
* **輸入暫存器 (`0x1000_0000` / `0x1000_0004`)**：映射至 16 個指撥開關 (Switches) 與確認按鈕 (btnC)。
* **輸出暫存器 (`0x1000_0008`)**：映射至顯示控制模式 (待機/PASS/Err) 與動態七段顯示器掃描電路。
